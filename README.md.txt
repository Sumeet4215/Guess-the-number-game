# 🎮 Number Guessing Game in Python

A fun and interactive number guessing game with three difficulty levels: **Easy**, **Moderate**, and **Extreme**. Built using simple Python logic with an engaging hint system that adapts to the difficulty selected.

---

## 🧠 About the Project

This game is designed to challenge users to guess a randomly generated number within a limited number of attempts (lives). It tests your logical thinking and gives helpful clues depending on the difficulty level.

---

## 💡 Features

- 🎯 **3 Difficulty Levels**:
  - **Easy**: Guess numbers between 0–9.
  - **Moderate**: Guess numbers between 0–20.
  - **Extreme**: Extra hints (odd/even) added to help you guess.

- ❤️ **3 Lives** to guess the number correctly.

- 🧩 Smart hints:
  - "Think Higher", "Think Lower"
  - In Extreme mode: "Think Lower and even", "Think Higher and odd", etc.

- 👨‍💻 Simple and clean code structure using conditionals, loops, and random module.

---
